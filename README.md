#  Volunteer Hub 

> **Connect. Engage. Impact.** > A bridge between passion and purpose in Bangladesh.

---

### The Vision
Volunteer Hub isn't just a project; it's a solution. Built for university students and local NGOs, it streamlines the journey from finding a cause to making a real-world difference.

###  Quick Features
* **Dual-Core Dashboards:** Custom experiences for both Volunteers and NGOs.
* **Cinematic UI:** A dark-mode, glassmorphic design inspired by premium streaming platforms.
* **Secure Core:** Powered by Firebase Auth and a custom Node/MongoDB backend.

### The Tech Toolbox
- **Logic:** Node.js + Express.js
- **Database:** MongoDB Atlas
- **Security:** Firebase Authentication
- **Interface:** HTML5, CSS3, Vanilla JS
